## 原始表达式



```
1.23       //数字直接量
"hello"    //字符串直接量
"pattern"  //正则表达式直接量
```



javascirpt中的一些保留字构成了原始表达式

```
true    //返回一个布尔值: 真
false   //返回一个布尔值：假
null    //返回一个值：空
this    //返回"当前"对象
```

this关键字经常在面向对象编程中出现

第三种原始表达式是变量

````
i         //返回变量的值
sum       //返回sum的值
undefined //undefined是全局变量，和null不同，它不是一个关键字
````

如果变量不存在，表达式运算结果为undefined。
